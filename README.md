VidSnapAI - Reel Generator 

VidSnap AI is a web application that generates AI-powered voice/video content using the ElevenLabs API.
This project was developed using Python and Flask with a simple and responsive frontend built using HTML and Bootstrap.

The application allows users to generate realistic AI audio using the ElevenLabs API and display the results through an easy-to-use interface.


Features:

AI voice generation using ElevenLabs API
Simple and responsive user interface
Fast API integration with Flask backend
Clean frontend built using Bootstrap
Easy to run locally
